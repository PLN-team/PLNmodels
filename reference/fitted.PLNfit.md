# Extracts model fitted values from objects returned by [`PLN()`](https://pln-team.github.io/PLNmodels/reference/PLN.md) and its variants

Extracts model fitted values from objects returned by
[`PLN()`](https://pln-team.github.io/PLNmodels/reference/PLN.md) and its
variants

## Usage

``` r
# S3 method for class 'PLNfit'
fitted(object, ...)
```

## Arguments

- object:

  an R6 object with class
  [`PLNfit`](https://pln-team.github.io/PLNmodels/reference/PLNfit.md)

- ...:

  additional parameters for S3 compatibility. Not used

## Value

A matrix of Fitted values extracted from the object object.
